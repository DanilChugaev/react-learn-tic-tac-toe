# Tic Tac Toe game

Простая игра в крестики-нолики, доработана из [туториала](https://react.dev/learn/tutorial-tic-tac-toe) по React 

[PLAY HERE](https://danilchugaev.github.io/react-learn-tic-tac-toe/)

## Run Locally

Clone the project

```bash
  git clone git@github.com:DanilChugaev/react-learn-tic-tac-toe.git
```

Go to the project directory

```bash
  cd react-learn-tic-tac-toe
```

Install dependencies

```bash
  yarn install
```

Start the development server

```bash
  yarn dev
```

Open http://localhost:5173/react-learn-tic-tac-toe/ in your browser (note: the /react-learn-tic-tac-toe/ base path is for GitHub Pages; adjust if needed locally)
